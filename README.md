# Practice Problems

### Getting Started

- Run `yarn install`
- Run `yarn start -p <PROBLEM_NUMBER> -t <TEST_NAME>`
  - Example `yarn start -p 1 -t a`
- Open a problem file from `src/problems/problem-*/problem-*.ts`
- Return solution in the default function

### Commands

- `yarn start`
  - `-p <PROBLEM_NUMBER>` (Required)
  - `-t <TEST_NAME>`
- `yarn format`
